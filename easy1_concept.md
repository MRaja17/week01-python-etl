## What is Python?

Python is a high-level, general-purpose programming language that is easy to read and write.

## How to Set Up Python

1. Install Python.
2. Check version:
   python --version
3. Create virtual environment:
   python -m venv .venv
4. Activate it:
   .venv\Scripts\activate  (Windows)
